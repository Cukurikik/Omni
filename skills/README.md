## 🚀 Cara Cepat Menggunakan

### Opsi 1 — Satu File (Paling Mudah)
Buka `meta/MASTER_SYSTEM_PROMPT.md`, copy semua isinya, paste ke System Instructions AI provider manapun.

### Opsi 2 — Per Skill (Modular)
Pilih skill yang dibutuhkan dari folder masing-masing. Cocok untuk AI agent dengan context window terbatas.

### Opsi 3 — Lihat INSTALL.md
Ikuti panduan instalasi khusus per provider (Claude, GPT, Gemini, dll).
