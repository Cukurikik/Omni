
# Architecture Deep Dive: omni-web-core

The underlying architecture of `omni-web-core` represents the pinnacle of unified language design in the OMNI Framework.

## The LLVM-Omni Compilation Pipeline
`omni-web-core` does not execute via traditional interpretation or simple JIT compilation like V8 (Node.js) or CPython. Instead, its source files are evaluated into a Language-Agnostic Abstract Syntax Tree (UAST) which is then recursively lowered into Intermediate Representation (IR).
